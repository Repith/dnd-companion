## Key Features

- **Character Management**: Full D&D 5e character creation and management
- **Campaign Tracking**: Quest and session management for DMs
- **Real-time Updates**: Event-driven architecture for live gameplay
- **Inventory System**: Equipment and item management
- **Spell Management**: Spell learning, preparation, and slot tracking
- **Generator Service**: AI-powered content generation (planned)

## Technology Stack

- **Backend**: NestJS, TypeScript, PostgreSQL, Prisma
- **Frontend**: Next.js 16, React 19, TypeScript, Tailwind CSS
- **Infrastructure**: Docker, Docker Compose, Kubernetes-ready
- **Testing**: Jest, React Testing Library, E2E tests
